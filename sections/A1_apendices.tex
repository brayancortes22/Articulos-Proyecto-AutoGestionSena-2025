\section{Herramientas de IA utilizadas}
Durante el desarrollo del proyecto ``Autogestión SENA'' se utilizaron las siguientes herramientas de Inteligencia Artificial:

\begin{itemize}[leftmargin=*]
  \item \textbf{Asistentes de código}: GitHub Copilot, ChatGPT para generación de código y debugging
  \item \textbf{Consultas técnicas}: Modelos de lenguaje para explicación de conceptos y validación de arquitecturas
  \item \textbf{Diseño y creatividad}: IA para inspiración de diseños de interfaz y búsqueda de patrones modernos
  \item \textbf{Aprendizaje}: Uso de IA como mentor técnico disponible 24/7 para el equipo
\end{itemize}

\section{Repositorio del proyecto}
El código fuente y la documentación completa del proyecto están disponibles para consulta académica. El proyecto incluye el modelo entidad-relación, mockups en Figma, código del back-end en Django/Python y el front-end web completo.
